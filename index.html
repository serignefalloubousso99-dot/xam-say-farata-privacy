Politique de confidentialité

Xam Say Farata est conçue pour être transparente, sécurisée et respectueuse de vos données. Aucun compte, email ou numéro de téléphone n'est requis pour utiliser l'application.

🔒

Données collectées

Nous ne collectons aucune donnée personnelle identifiable. Voici précisément ce qui est stocké :

  • Prénom saisi : stocké uniquement sur votre téléphone, jamais transmis.
  • Progression quiz (XP, rang, historique) : stockée localement sur votre téléphone.
  • Préférences (notifications, apparence, horaires) : stockées localement.
  • Identifiant Battle anonyme : un code aléatoire (ex: p_abc123) généré automatiquement, non lié à votre identité réelle, utilisé uniquement pour les parties en ligne.
👁️

Localisation

L'accès à votre position GPS est utilisé exclusivement pour calculer les heures de prière et la direction de la Qibla. Ce calcul s'effectue directement sur votre téléphone. Votre position n'est jamais transmise, partagée ou sauvegardée en dehors de votre appareil.

🔔

Notifications

Les notifications servent à deux usages :

  • Rappels de prière : programmés localement sur votre téléphone, sans serveur externe. Vous pouvez les désactiver à tout moment dans les paramètres.
  • Annonces et rappels de l'équipe : si vous acceptez les notifications, un identifiant technique anonyme (jeton de notification), le type d'appareil (Android/iOS) et la date de dernière activité sont enregistrés sur notre service Supabase. Ce jeton ne contient aucune information personnelle et sert uniquement à vous envoyer ces messages.
  • Ce jeton est supprimé si vous désactivez les notifications ou désinstallez l'application.
⚔️

Mode Battle (multijoueur)

Lorsque vous utilisez le Mode Battle, les données suivantes sont envoyées temporairement sur nos serveurs :

  • Pseudo choisi librement par vous (pas votre vrai nom).
  • Votre score et vos réponses durant la partie.
  • Votre identifiant Battle anonyme (code aléatoire non lié à votre identité).
  • Ces données sont automatiquement supprimées à la fin de chaque partie. Aucune donnée personnelle (email, téléphone, localisation) n'est jamais transmise.
🔐

Sécurité des données

Toutes les communications entre l'application et nos serveurs sont chiffrées via HTTPS. Nous utilisons Supabase, une plateforme certifiée SOC 2 qui stocke les données dans des centres de données sécurisés. Les données sur votre téléphone sont protégées par la sécurité système d'Android ou iOS.

🖥️

Services tiers

L'application utilise Supabase pour les usages suivants uniquement :

  • Mise à jour du contenu islamique (questions de quiz, cours, annonces) — aucune donnée personnelle transmise.
  • Mode Battle en temps réel — pseudo anonyme, score et identifiant de partie uniquement.
  • Envoi des notifications — jeton de notification anonyme uniquement.
🗑️

Vos droits et suppression des données

  • Supprimer toutes vos données locales : allez dans Paramètres → "Réinitialiser l'application".
  • Données Battle : supprimées automatiquement à la fin de chaque partie.
  • Jeton de notification : supprimé si vous désactivez les notifications ou désinstallez l'application.
  • Pour toute demande concernant vos données, contactez-nous via la page "Nous contacter" dans l'application.
  • Vous avez le droit d'accéder à vos données, de les corriger ou de les supprimer à tout moment.
